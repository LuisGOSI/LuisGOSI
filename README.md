# 👨‍💻 Luis Gonzalez | Desarrollador de Software & Cloud Enthusiast

¡Hola! Soy Luis, un apasionado desarrollador de software con una sólida base técnica y un enfoque en la creación de soluciones eficientes y escalables. Como TSU en TI y actual estudiante de Ingeniería en Desarrollo de Software en la [Universidad Tecnológica de León (UTL)](https://www.utleon.edu.mx/), me encuentro en mi 11vo cuatrimestre realizando prácticas profesionales.

Mi misión es crecer en el ecosistema del desarrollo de software, especializándome en el **back-end** y en tecnologías de **cloud computing**, para colaborar en proyectos desafiantes que me permitan aprender de otros desarrolladores y aportar soluciones de alto impacto.

---
## 🚩 Objetivo Profesional

Diseñar, implementar y desplegar funcionalidades y herramientas de software robustas utilizando tecnología de vanguardia. Mi objetivo es construir servicios web escalables y de alto rendimiento, aprovechando al máximo las ventajas de la nube para mejorar la eficiencia, confiabilidad y funcionalidad de los proyectos en los que participo.

---
### 📘 Resumen Profesional

Soy un entusiasta del desarrollo de software con un fuerte impulso por aprender y aplicar las últimas tecnologías. Mi pasión se centra en el desarrollo backend y la arquitectura en la nube, pero mi curiosidad me ha llevado a explorar áreas como la **Inteligencia Artificial**, el **IoT** y el **desarrollo de videojuegos**. Estoy siempre abierto a colaborar, compartir conocimientos y enfrentar nuevos retos.

### 🛠 Proyectos Destacados

1.  **CySe-Net**: Página web corporativa con chatbot integrado para una microempresa de servicios en San Juan del Río, Qro.  
    [🔗 Ver proyecto](https://computacionyservicio.mx)
2.  **DonGalleto**: Aplicación integral con múltiples módulos de gestión administrativa y operativa para una empresa galletera.  
    [💻 Código fuente](https://github.com/LuisGOSI/donGalleto)
3.  **RememberMe**: Aplicación móvil que se comunica vía Bluetooth con un módulo ESP32 para el recordatorio de medicamentos.  
    [💻 Código fuente](https://github.com/LuisGOSI/rememberMeExpo49)
4.  **Qronecta**: Plataforma web/móvil diseñada para reducir la brecha digital entre la ciudadanía y los servicios de una ciudad inteligente.  
    [💻 Código fuente](https://github.com/LuisGOSI/Qronecta-Hackhaton)
5.  **QroMovil**: Aplicación móvil enfocada en mejorar la experiencia de los servicios de transporte público en el estado de Querétaro.  
    [💻 Código fuente](https://github.com/victoMR/QroMovil)
6.  **FilmBox**: Aplicación web para consultar y publicar reseñas de películas.  
    [💻 Código fuente](https://github.com/LuisGOSI/FilmBox)
7.  **GreenWaste**: Aplicación web que promueve la gestión responsable de residuos a través de manualidades y consejos prácticos.  
    [💻 Código fuente](https://github.com/LuisGOSI/GreenWaste_Pagina)
